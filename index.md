# Index

Welcome to TianoCore, the community supporting an open source implementation of [UEFI](UEFI.md). [EDK II](EDK-II.md) is a modern, feature-rich, cross-platform firmware development environment for the UEFI and [PI](PI.md) specifications. We hope that you’ll delve into our work, use TianoCore for platform firmware, [report any issues](Reporting-Issues.md) you find, and contribute to the community. Learn more on [TianoCore Who we are](TianoCore-Who-we-are.md).

The latest [stable tag](https://github.com/tianocore/tianocore.github.io/wiki/EDK-II#stable-tags) of EDK II is [edk2-stable202508](https://github.com/tianocore/edk2/releases/tag/edk2-stable202508).

If you want to compile firmware or utilities, we recommend the [Getting Started with EDK II](Getting-Started-with-EDK-II.md) page. This provides an overview of downloading [EDK II from github](https://github.com/tianocore/edk2) and building a sample platform ([OVMF](OVMF.md), [EmulatorPkg](EmulatorPkg.md), [MdePkg](MdePkg.md), ...). There are multiple [hardware platforms](EDK-II-Platforms.md) using EDK II open source UEFI firmware, including [MinnowBoard Max/Turbot](MinnowBoardMax.md), [Aaeon UpSquared](IntelAtomProcessorE3900.md), and [Intel® Galileo Gen 2](Galileo.md). This allows you to experiment and develop UEFI features on real hardware.

TianoCore uses github issues to track issues and feature requests. Please review [Reporting Issues](Reporting-Issues.md) and [Reporting Security Issues](Reporting-Security-Issues.md) for more information. If you would like to contribute code or fix issues, please see [How To Contribute](How-To-Contribute.md) along with our [Inclusive Language Guidelines](Inclusive-Language-Guidelines.md). The [Tasks](Tasks.md) page has a list of priority work items.
